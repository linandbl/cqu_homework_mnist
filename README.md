# 文件说明
## .idea
pycharm配置文件
## dataset
MNIST数据集，模型中已设置好路径。若丢失请在datasets.MNIST中设置download=True重新下载

## env
环境配置文件
conda环境配置可通过以下命令导入
```bash
conda env create -f environment.yml
conda create -n new_env_name --file conda_requirements.txt
```
## model
训练好的的模型以及性能参数

## cnn_munal.py
手写cnn模型

## cnn_pytorch.py
pytorch框架下cnn模型
默认采用cpu训练
